# TaskmanagerRestAPI
Task Management with mongodb,nodejs

Mongoose :
npm i mongoose@5.3.16

Validator:
npm i validator@10.9.0

REST API : Structure :
Create: POST /tasks

Read: GET /tasks
Read: GET /tasks/:id

Update:
PATCH /tasks/:id

Delete:
delete /tasks/:id

# nodemon
npm i nodemon@1.18.9 --save-dev
